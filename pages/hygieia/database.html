Database Setup for Supported Platforms

Hygieia uses MongoDB (versions 3.0 and above) as the database to store and retrieve data. Download any MongoDB GUI client, such as Robo 3T, to create the database dashboard and connect to it.

  • Windows
  • MAC
  • Linux

Download and Install Instructions

If you do not already have MongoDB installed, download the Installation Package from MongoDB download page and follow the installation instructions provided in the MongoDB manual.

Configuration Steps

Follow the configuration steps to run Hygieia in your MongoDB installation.

  • Step 1: Create Data Directory

    • Create a data directory on the drive you use to start MongoDB to store your data files.

    Create the data directory (the C drive is used in this case):

    C:\Users\[usernname]\dev\data\db
  • Step 2: Change Directory

    • Change the current working directory to the bin directory of your MongoDB installation.

    In the command prompt, run th following command:

    cd C:\Program Files\MongoDB\Server\3.2\bin
  • Step 3: Start MongoDB

    • Specify the path to use as data directory:

    mongod --dbpath [path to the data directory]

    Execute the following commands to start MongoDB, switch to dashboard with name dashboarddb, and then add a dashboard user:

    #Start mongo

Command: > mongo
Output:  MongoDB shell version: 3.2
		 connecting to: test
			
#Switch to db dashboard

Command: > use dashboarddb
Output:  switched to db dashboard

#Create db user

Command: > db.createUser(
			{
			  user: "dashboarduser",
			  pwd: "dbpassword",
			  roles: [
			  {role: "readWrite", db: "dashboarddb"}
			  ]
			})
			
Output:  Successfully added user: {
         "user" : "dashboarduser",
         "roles" : [
		  {
			"role" : "readWrite",
			"db" : "dashboarddb"
		  }
		  ]
		  }

Executing through a Script

To execute the above via a script in an automated fashion, use mongosrc.js.

 mongo < mongosrc.js
